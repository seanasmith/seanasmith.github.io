Course Description:

This course provides students fundamentals in designing and maintaining content on the web through hand-coded websites, provided frameworks and content management systems. Students will also discuss theoretical approaches to design, user research, design thinking and web semantics.

Learning Objectives:

  • Building websites using HTML and CSS
  • Building websites using CSS Framework Bootstrap
  • Build websites using content management systems (Wordpress)
  • Discuss design and how it affects interactions on the web.
  • Learn to conduct user research and apply that to design
  • Learn web journalism techniques and how to effectively write and design content for the web

Quizzes and Exercises

Outside of the major projects in the class, students will be assessed using a variety of quizzes and exercises. A total of five quizzes are spaced throughout the semester and are worth 20 points each. There are four execrises scheduled during the semester. Each exercise is worth 25 points, totalling 200 points when combined with the quizzes.

Quizzes

  • Quiz 1: HTML
  • Quiz 2: CSS
  • Quiz 3: Bootstrap
  • Quiz 4: Multimedia Journalism
  • Quiz 5: Wordpress

Exercises

  • Troubleshooting exercise
    • Correct errors in provided code
  • iMovie Exercise
    • Create a short video with interview edited in iMovie
  • Responsive exercise
    • Add media queries to code for mobile optimization

    • Each student will be assigned a day to present a tech news item to the class

Policy on late work:

Late work will not be accepted. The professional environments in which web developers and web journalists work often involve strict deadlines and rigorous timelines. This class will follow that concept. In cases where the student believes he or she deserves an extension on a project, it is his or her responsibility to contact the Dean of Students' office and launch the necessary formal steps to gain an extension.

Attendance

Attendance is required for all classes and workdays. You are allowed four absences with no penalty throughout the semester. Any absences following the fourth will result in a 2 percent deduction from your overall grade.

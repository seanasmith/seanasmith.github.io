Project 2: Organziation Site

Learning Objectives

  • Conduct basic user research
  • Build a site from design sketches to functional product
  • Practice responsive design and using CSS frameworks

Description:

You will build a website for an organization of your choice using CSS framework Bootstrap. You will conduct basic user research, draft early concepts of the site and design according to the organization's needs. This site will also require responsive design that optimizes the site for mobile and tablet viewing.

Steps:

  1. Conduct your user research. Interview at least two people using the basic user research activity.
  2. Sketch two drafts of the site you will design. Draw on what you learned from the user research to guide your design.
  3. Design the site using CSS framework Bootstrap.
  4. Optimize the site for mobile and tablet using responsive design elements within your HTML and CSS.
    • Use breakpoints from the Chrome Inspector at Mobile L and Tablet
  5. You will present the site to the class on the due date and explain how the user research inspired your design and how you incorperated those findings in the site.

Grading:

Project 2 is worth 150 points. See the rubric for detailed breakdown of grading criteria.

  • A+: Clean code, advanced Bootstrap components used correctly, design heavily influenced by user research.
  • A: Influenced by user research, few errors in code, decent variety of Bootstrap components used.
  • B: Little influence from user research, minimal Bootstrap, some responsive design issues.
  • C: Several missing components, ineffective influence from user research, responsive design issues, coding issues.
  • F: Systemic errors, turned in late, critical errors.
